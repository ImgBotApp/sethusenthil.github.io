This "hack" was not based on meta refresh, this pages was iframed in one of the boxes, and this website you are currently viewing is equipped with an iframe breaking/hijacking script.


iframe hijacking script:

if (top.location!= self.location) { top.location = self.location.href; }

How to patch this annoying iframe vulnerability?

It's pretty simple, next time you iframe I recommend you sandbox it. It should look something like:

iframe sandbox="allow-same-origin allow-scripts allow-popups allow-forms" src="https://www.example.com"
